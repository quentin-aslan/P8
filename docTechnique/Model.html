    read(queryopt, callbackopt)
    model.js, line 41

    Recherche et retourne une tâche, si le paramètre "query" est vide alors toutes les taches seront renvoyé. Si une chaîne ou un nombre, il le recherchera.
    Parameters:
    Name Type Attributes Description
    query string | number | object <optional>
    		 Requête pour filtrer les tâche à recupérer
    callback function <optional>
    		 Fonction de callback appelée lorsque la tâche est trouvé
    Example
    model.read(1, func); // Will find the model with an ID of 1
model.read('1'); // Same as above
//Below will find a model with foo equalling bar and hello equalling world.
model.read({ foo: 'bar', hello: 'world' });
